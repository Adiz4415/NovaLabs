| Version | Supported          |

## Reporting a Vulnerability

We take security vulnerabilities seriously. Please report them via **GitHub Security Advisories**:

1. Go to https://github.com/NovaCoreLabs1/NovaLabs/security/advisories
2. Click "New draft security advisory"
3. Fill in the details — include steps to reproduce, affected versions, and potential impact

You can expect an acknowledgment within 48 hours and a resolution timeline within 14 days.

For sensitive disclosures, contact the security team directly at the email address listed in the GitHub security tab.

We appreciate your help in keeping NovaLabs and its users safe.
